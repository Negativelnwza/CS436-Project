## Design Rationale

- Single page: ลดความซับซ้อนและตรงตามโจทย์
- Two-line display: expression + result — ช่วยให้ผู้ใช้เห็นสิ่งที่พิมพ์และผลลัพธ์ชัดเจน
- ปุ่มขนาดใหญ่: เอื้อต่อการใช้งานบนมือถือ
- สี: ใช้สีฟ้าสำหรับปุ่ม = และสีเทาอ่อนสำหรับพื้นหลัง (เพื่อให้ Contrast สูงและอ่านง่าย)
- รองรับคีย์บอร์ด: เพิ่มความสะดวกสำหรับผู้ใช้เดสก์ท็อป
- การรองรับ operator precedence: เพื่อผลลัพธ์ที่ถูกต้องตามคาดหมาย (เช่น 2 + 3 * 4 = 14)
