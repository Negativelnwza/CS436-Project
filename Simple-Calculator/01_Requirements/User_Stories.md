## User Stories

1. ในฐานะผู้ใช้ ฉันต้องการกดตัวเลขบนหน้าจอเพื่อป้อนค่าที่ต้องการคำนวณ
2. ในฐานะผู้ใช้ ฉันต้องการกดปุ่มบวก ลบ คูณ หาร เพื่อสร้าง expression สำหรับคำนวณ
3. ในฐานะผู้ใช้ ฉันต้องการกดปุ่ม = เพื่อดูผลลัพธ์ของ expression
4. ในฐานะผู้ใช้ ฉันต้องการกด C เพื่อเคลียร์ค่าและเริ่มใหม่
5. ในฐานะผู้ใช้ ฉันต้องการกด ⌫ (Backspace) เพื่อลบตัวอักษรสุดท้ายเมื่อเผลอพิมพ์ผิด
6. ในฐานะผู้ใช้ ฉันต้องการป้อนทศนิยม (.) เพื่อคำนวณตัวเลขทศนิยม
7. ในฐานะผู้ใช้ ฉันต้องการให้ระบบแจ้งเตือนหรือจัดการเมื่อฉันพยายามหารด้วยศูนย์
8. ในฐานะผู้ใช้ ฉันต้องการสามารถใช้คีย์บอร์ด (ตัวเลข และ + - * / Enter Backspace) ได้
9. ในฐานะผู้ใช้ ฉันต้องการให้ผลลัพธ์แสดงอย่างถูกต้องตามลำดับการคำนวณ (operator precedence)
10. ในฐานะผู้ใช้ ฉันต้องการ UI ที่ชัดเจนและใช้งานง่ายทั้งบนมือถือและเดสก์ท็อป
